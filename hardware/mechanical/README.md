# Diseño mecánico

Diseños 3D, piezas impresas, construcción manual y ensamblaje.

## Contenido esperado

- Archivos STL para impresión 3D
- Archivos fuente (OpenSCAD, Tinkercad links)
- Archivos GCode con parámetros de impresión
- Instrucciones de ensamblaje
- Fotos del proceso de construcción

## Del equipo 2025

El equipo 2025 usó Tinkercad y OpenSCAD para los diseños 3D. Incluye soportes de motores TT, cilindros para dribbler, estructura de pisos, etc.

Los archivos binarios originales (STL, GCode) se encuentran en el [repositorio 2025](https://github.com/IITA-Proyectos/RoboCupJunior-Soccer-Open-League-2025/tree/main/Dise%C3%B1o3D).

Los archivos fuente OpenSCAD están migrados en `legacy/2025-season/mechanical/`.
