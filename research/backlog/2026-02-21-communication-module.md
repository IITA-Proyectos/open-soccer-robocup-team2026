# Integración del Communication Module

Obligatorio en competencia internacional.

## Requisitos técnicos

- Pin GPIO 2.54mm para start/stop
- Alimentación directa a batería (5.3V-25V) sin switch, 500mA
- Siempre encendido, incluso robot fuera del campo
- ≥1cm del borde exterior, protegido de impactos
- No cuenta para peso ni altura
- Repo: https://github.com/robocup-junior/soccer-communication-module
