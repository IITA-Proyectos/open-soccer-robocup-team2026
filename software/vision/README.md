# Visión - OpenMV

Código de la cámara OpenMV (H7 / H7 Plus) para detección de pelota y arcos.

## Punto de partida

Ver el código de visión 2025 en `legacy/2025-season/code/vision-openmv/` como referencia.

## Funcionalidades

- Detección de pelota por color (thresholds LAB)
- Detección de arcos (1 o 2 arcos)
- Cálculo de coordenadas
- Envío de datos por UART al Teensy
- Calibración de thresholds
